Le projet Groupie-Tracker

Groupie-Tracker est un projet dont le but est de récupérer les données d'un API json en ligne et de styliser le tout dans une page web. Pour arriver à cela, nous avons décidé de récupérer l'API à l'aide d'un programme en Go qui va ordonner les données selon une structure et créer un template html qui peut ensuite être intégré à la page.
